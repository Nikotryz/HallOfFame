# HallOfFame

## Основные задачи
- Приведение БД к коду с помощью миграций;
- Все основные End-points:
    1. GET api/v1/persons
    2. GET api/v1/persons/{id}
    3. POST api/v1/persons
    4. PUT api/v1/persons/{id}
    5. DELETE api/v1/persons/{id}
- Разные статусы ответов для End-points.

## Дополнительные задачи
- Корректная обработка ошибок пользовательского ввода;
- Покрытие бизнес-логики интеграционными и Unit-тестами;
- Логгирование работы приложения в консоль;
- Логгирование ошибок при работе в физический файл '.txt';
- Возможность работы приложения в Docker-контейнере.

## Начало работы (с помощью Docker)
- Склонировать репозиторий: 'git clone https://github.com/Nikotryz/HallOfFame';
- Открыть консоль в директории скачанного репозитория;
- Ввести команду 'docker-compose build'. Начнется сборка приложения, дождитесь окончания;
- Ввести команду 'docker-compose up -d' для запуска приложения;
- Открыть в браузере [http://localhost:8080/swagger/index.html](url).
